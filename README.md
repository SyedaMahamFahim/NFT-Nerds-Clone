## Tech Stack
- React
- MUI
- Framer Motion


## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/NFT-Nerds-Clone.git
```

Go to the project directory

```bash
  cd nft-nerds-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start
```
